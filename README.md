# Post-Discharge Nephrology AI Assistant

A multi-agent AI chatbot system that helps patients after hospital discharge, focusing on nephrology (kidney disease) care.

## Features

- **Multi-Agent Architecture**: Receptionist + Clinical agents
- **Patient Data Retrieval**: 25+ dummy patient records
- **RAG System**: Nephrology knowledge base
- **Web Search**: Current medical information
- **Streamlit Frontend**: Easy-to-use interface
- **Comprehensive Logging**: All interactions tracked
